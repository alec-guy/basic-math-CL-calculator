# gadts

#### a command line calculator 

This calculator supports 

- sin 
- cosine 
- tangent 
- sqrt 

and normal arithmetic

   ```text 
   =Calculator=
   :22 + 21
   Expression = (22.0) + (21.0)
   Evaluation = 43.0
   :(29 / 24) + 21
   Expression = ((29.0) / (24.0)) + (21.0)
   Evaluation = 22.208333333333332
   :!
   Thank you for using this calculator
   ```
